# Boardly 🏷️📌

**Boardly** is a social media platform inspired by Pinterest, allowing users to create, share, and organize pins and boards. It features AI-powered keyword suggestions for images, trending feeds, and a smooth user experience.  

Watch the full demo here: [https://youtu.be/uXOOn0DeI6U](https://youtu.be/uXOOn0DeI6U)

The site is hosted live at: [https://boardly-social-media.vercel.app/](https://boardly-social-media.vercel.app/)

---

## Features ✨

- **User Authentication**: Sign up, log in, and log out securely.
- **Create Pins**: Upload images with titles, descriptions, and keywords.
- **AI Keyword Suggestions**: Generate SEO-friendly tags from uploaded images using OpenAI’s vision-enabled model.
- **Boards**: Organize pins into custom boards.
- **Trending Section**: Discover the most popular pins.
- **Drag & Drop Upload**: Simple and intuitive image upload experience.
- **Responsive Design**: Works seamlessly on desktop and mobile.

---

## Tech Stack 🛠️

- **Frontend**: React, TailwindCSS, Axios, React Router
- **Backend**: FastAPI, Python, SQLAlchemy
- **Database**: PostgreSQL / SQLite
- **AI Integration**: OpenAI GPT-4o-mini for image-based keyword generation
- **Hosting**: Frontend on Vercel, Backend on Render
- **Real-time Features**: WebSocket support for live updates
